## Definition

Virtual Delivery Agent là agent được cài trên máy desktop hoặc server dùng để cung cấp phiên Citrix, giúp máy đó đăng ký với Controller và truyền phiên HDX tới người dùng.

## Variants

- VDA cho desktop OS.
- VDA cho server OS hoặc RDS workload.
