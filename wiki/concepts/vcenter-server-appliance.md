## Definition

vCenter Server Appliance là appliance dựa trên Linux dùng để triển khai vCenter Server và các dịch vụ quản trị đi kèm.

## Variants

- Triển khai bằng giao diện đồ họa.
- Triển khai bằng CLI hoặc quy trình tự động hóa.
