## Definition

Virtual machine là máy tính ảo chạy trên hypervisor, có CPU, memory, disk, network adapter và hệ điều hành khách.

## Variants

- Desktop VM cho người dùng VDI.
- Server VM cho dịch vụ hạ tầng hoặc RDS workload.
- Template hoặc gold image dùng để tạo nhiều VM.
