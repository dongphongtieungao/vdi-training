## Definition

vCenter Server là thành phần quản trị tập trung cho vSphere, dùng để quản lý ESXi host, cluster, VM, datastore, network, quyền truy cập và tích hợp với nền tảng VDI.

## Variants

- vCenter Server Appliance.
- Nhiều vCenter trong Enhanced Linked Mode.
