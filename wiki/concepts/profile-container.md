## Definition

Profile container là tệp container chứa hồ sơ người dùng, được gắn vào phiên Windows để người dùng có cùng dữ liệu cá nhân trên nhiều máy VDI hoặc session host.

## Variants

- Container VHD hoặc VHDX.
- Container đặt trên SMB share hoặc nền tảng file storage được hỗ trợ.
