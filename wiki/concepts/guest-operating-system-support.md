## Definition

Guest operating system support là danh sách hệ điều hành khách được nền tảng hypervisor hỗ trợ chính thức, gồm driver, tool, tính năng và giới hạn tương thích.

## Variants

- Windows desktop hoặc server cho VDI.
- Linux hoặc hệ điều hành khác cho workload phụ trợ.
