## Definition

Machine identity là cách một máy desktop hoặc server trong VDI được định danh với domain, directory hoặc mô hình không joined domain, ảnh hưởng đến xác thực, chính sách và quản lý vòng đời máy.

## Variants

- Active Directory joined.
- Microsoft Entra hybrid joined.
- Non-domain-joined.
